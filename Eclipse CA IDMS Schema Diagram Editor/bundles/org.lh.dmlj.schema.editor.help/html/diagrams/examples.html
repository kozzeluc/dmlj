Examples

To checkout whether the diagram editor can be of any use to you, download the following examples and copy them to a project in your Eclipse workspace (refresh the project to make sure they are visible in the Package Explorer view or paste them directly in a project in the Package Explorer view) :
  • EMPSCHM version 100
  • IDMSNTWK version 1 (Release 19.0)
Then, open them and start playing with the diagram editor.
